The example shows how to integrate C# and Python using the new dynamic feature found in C# 4.0. To run the sample, you will need to install IronPython 2.0, which is a available as a separate download. It may also work with more recent versions of IronPython. After installing IronPython you will need to update this sample by replacing the IronPython, IronPython.Modules, Microsoft.Scripting, and Microsoft.Scripting.Core assemblies found in the References section. Open the Solution Explorer and right click on the References node. Remove the assemblies listed earlier in this paragraph. Choose Add Reference. Browse to the directory where you installed IronPython and add the deleted assemblies. By default, IronPython is installed in the Program Files directory, so you should be able to find the assemblies there. After you have updated your References section, the program should compile and run without error.
